# TRONK To-Do List — План реализации

## Стек технологий

| Часть | Технология |
|---|---|
| Фронтенд | Nuxt 3 + Vue 3 + TypeScript |
| Стейт | Pinia |
| Стили | Tailwind CSS + Bootstrap Icons |
| HTTP-клиент | Axios |
| Бэкенд | Node.js + Express (serverless на Vercel) |
| Авторизация | JWT (jsonwebtoken) |
| БД | Neon (PostgreSQL) |
| ORM | Prisma |
| Деплой | Vercel (фронт + бэк) |
| Доп. пакеты | bcrypt, zod, @vueuse/core, vee-validate |

---

## UI/Макет (из скриншотов)

### Страница логина
- Заголовок "To-Do List" + подзаголовок "Управляйте задачами эффективно"
- Поля: Email, Пароль, чекбокс "Запомнить меня"
- Кнопка "Войти" (синяя, полная ширина)
- Ссылка "Забыли пароль?"

### Страница списка задач
- Заголовок + кнопка "+ Добавить задачу" (верхний правый угол)
- Табы: Все задачи / Активные / Выполненные
- Строка поиска + кнопка "Очистить" + дропдаун сортировки
- Сортировка: По дате (возр.), По дате (убыв.), По названию, По приоритету
- Таблица (desktop): чекбокс, Название, Приоритет, Дата, Автор (аватар+имя), действия
- Список (mobile): компактный вид с бейджами
- Пагинация: ← 1 2 ... 4 5 →

### Приоритеты (бейджи)
- **Сегодня** — оранжевый бейдж (дедлайн сегодня)
- **Важно** — синий бейдж
- **Обычный** — без бейджа или серый

### Детальная страница задачи (правая панель / отдельная страница)
- "← Назад" + кнопки "Редактировать" / "Удалить"
- Заголовок задачи + бейдж приоритета
- Аватар автора + имя + email
- Секция "Описание"
- Дата и Приоритет

### Модальное окно "Новая задача"
- Поля: Название задачи, Описание, Дата (datepicker), Приоритет (select)
- Кнопки: "Отмена" / "Добавить"

### Toast-уведомления
- Зелёный: "Задача успешно добавлена!" (с иконкой галочки)

## Модель данных (обновлённая)

```
User: id, email, password, role, name, avatar?
Task: id, title, description, dueDate, isCompleted, priority, createdAt, userId
```

Поле `priority`: `NORMAL | IMPORTANT` (Сегодня вычисляется динамически по dueDate)
